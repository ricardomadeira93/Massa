# Massa

A multilingual cookie ordering landing page built with Next.js.

## Live App

https://massa-ashen.vercel.app/

## What I Built

1. Responsive landing page for a local food business.
2. Multilingual interface with Portuguese, English, Spanish, Italian, and Arabic.
3. WhatsApp based ordering flow.
4. Product cards, FAQ, pricing sections, and mobile first layout.

## Stack

Next.js, TypeScript, Tailwind CSS, Vercel
